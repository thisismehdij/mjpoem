# mjpoem Sitemap

این ریپازیتوری برای نگهداری و انتشار فایل‌های sitemap سایت **mjpoem** ساخته شده است.  
هدف از این پروژه معرفی دسته‌بندی‌ها و صفحات مهم سایت به موتورهای جستجو (مثل گوگل) است تا ایندکس شدن سریع‌تر و دقیق‌تر انجام شود.

## محتویات
- `sitemap.xml` : فایل اصلی شامل دسته‌بندی‌ها و صفحات مهم
- (در آینده می‌توان فایل‌های sitemap دیگری هم اضافه کرد)

## نحوه استفاده
آدرس عمومی این فایل پس از فعال‌سازی GitHub Pages در دسترس خواهد بود و می‌توان آن را در Google Search Console معرفی کرد.
